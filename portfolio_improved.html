The Problem

During the 2024 election cycle, I observed that graphic designers were spending 1-5 minutes per file search in our Dropbox system. With hundreds of daily file requests, this inefficiency was costing the team hours of productive time weekly. Our existing file system lacked semantic understanding of political content, making it impossible to search by campaign themes, candidate messaging, or visual elements.

The Solution

I designed and built a full-stack AI-powered catalog system that automatically analyzes, tags, and indexes political design files. The application integrates with existing Dropbox workflows while providing intelligent search capabilities that understand political context and visual content.
